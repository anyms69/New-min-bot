# Туристическое приложение "Русские просторы"

Красивое веб-приложение для отображения туристических экскурсий по России.

## Функционал

- ✅ Современный адаптивный дизайн
- ✅ Навигация между разделами
- ✅ Каталог экскурсий с фильтрацией по регионам
- ✅ Детальная информация о каждом туре
- ✅ Контактная форма
- ✅ Мобильная версия
