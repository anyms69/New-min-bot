# Туристическое приложение "РусскийТур"

Полнофункциональное веб-приложение для отображения туристических туров по России с 25+ направлениями.

## Особенности

- ✅ 25+ готовых туров по всем регионам России
- ✅ Современный адаптивный дизайн
- ✅ Темная/светлая тема
- ✅ Работающая навигация с 5 разделами
- ✅ Поиск и фильтрация туров
- ✅ Интерактивная карта регионов
- ✅ Детальная информация о каждом туре
- ✅ Анимации и плавные переходы
- ✅ Мобильная версия
- ✅ Готово для подключения реальных данных

## Технологии

- HTML5, CSS3 (Grid, Flexbox, CSS Variables)
- Vanilla JavaScript (ES6+)
- Swiper.js для слайдера
- Font Awesome иконки
- Google Fonts (Inter, Playfair Display)
